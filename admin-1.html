Cara Memasang Bridge Service

  1. Download paket Bridge Service (file bridge-service.zip) — saya akan kirimkan file ini terpisah.
  2. Ekstrak ke komputer yang satu jaringan dengan SQL Server 2008.
  3. Edit bridge.config.json: isi host, nama database, user, dan password SQL Server Anda.
  4. Jalankan start-bridge.bat (Windows). Jendela hitam akan tetap terbuka selama bridge aktif.
  5. Catat alamat yang muncul, misal http://192.168.1.10:4501, masukkan ke kolom "Alamat Bridge Service" di atas.
  6. Klik "Test Koneksi" untuk memastikan berhasil, lalu atur tabel tujuan dan jalankan sync.

Radius Lokasi Absen

Koordinat kantor sudah ditetapkan dan tidak dapat dipindah dari peta. Anda dapat mengubah nama kantor dan radius toleransi jarak absen.
